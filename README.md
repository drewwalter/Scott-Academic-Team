# Academic Team Study Materials

This is study Materials for all subjects in the KAAC Governors Cup Academic Compotition. There are Quizlets, Blookets and Internet links to materials as well as PDFs.

## Subjects

- [**Arts & Humanities**](Arts%20&%20Humanities)
- [**Language Arts**](Language%20Arts)
- [**Math**](Math)
- [**Quick Recall**](Quick%20Recall)
- [**Science**](Science)
- [**Social Studies**](Social%20Studies)

## Usage

I have created this folder structure based on the subjects that are on the KAAC website and test quizzes. To use this repository, simply navigate to the subject you want to study and click on the link or file that you want to use. If it is a file, please make a copy of it before using it (If useing the main repository).

## Contributing

I have tried to make this repository as complete as posable but I will have inevitably forget or misrepresent something, so in that case if you have any materials that you would like to contribute to this repository, please follow these steps:

1. Open [Github](https://github.com/drewwalter/Scott-Academic-Team) (If on Google Drive or File Explorer)
2. Fork this repository.
3. Create a new branch for your contribution.
4. Add your materials to the appropriate subject folder.
5. Commit your changes and push them to your forked repository.
6. Create a pull request to merge your changes into the main repository.
